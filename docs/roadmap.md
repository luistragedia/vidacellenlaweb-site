# Roadmap - VidaCell en la Web

## Fase 1 - Web base

- Crear estructura inicial.
- Conectar proyecto a GitHub.
- Diseñar landing principal.
- Agregar información básica de la marca.
- Preparar SEO inicial.

## Fase 2 - Marca y contenido

- Definir identidad visual.
- Crear secciones para clientes.
- Crear secciones para talleres.
- Añadir canales de contacto.

## Fase 3 - Catálogo simple

- Mostrar productos destacados.
- Añadir consultas por WhatsApp.
- Validar demanda antes de implementar tienda completa.

## Fase 4 - Sistema de talleres

- Crear subdominio talleres.vidacellenlaweb.com.
- Diseñar panel básico de órdenes.
- Registrar clientes, equipos, diagnósticos y estados.
- Evaluar uso por talleres aliados y subtalleres.

## Fase 5 - Plataforma

- Catálogo avanzado.
- Aliados comerciales.
- Control de pedidos.
- Panel administrativo.
- Integraciones de pago y logística.
