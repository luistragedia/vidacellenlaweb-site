# VidaCell en la Web

Sitio web principal de VidaCell en la Web.

## Objetivo

VidaCell en la Web será una plataforma digital orientada a tecnología, comercio online, servicios técnicos y futuras soluciones para talleres, clientes y aliados comerciales.

## Estructura inicial

- `index.html`: página principal.
- `assets/css/style.css`: estilos del sitio.
- `assets/js/main.js`: scripts básicos.
- `assets/img/`: imágenes y recursos visuales.
- `docs/roadmap.md`: planificación futura del proyecto.
- `.cpanel.yml`: configuración de despliegue hacia cPanel.

## Fases previstas

1. Landing principal.
2. Identidad visual y contenido base.
3. Catálogo simple.
4. Subdominio para sistema de talleres.
5. Plataforma comercial/digital más avanzada.

## Estado

Proyecto en fase inicial.
